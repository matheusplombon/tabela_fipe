# Projeto de Consumo da Tabela Fipe em Java

---

## 🇧🇷 Português

Este projeto foi desenvolvido como um desafio da Alura para praticar o consumo de APIs em Java, utilizando a Tabela Fipe como fonte de dados. Ele demonstra como realizar requisições HTTP, manipular respostas JSON e integrar esses dados em uma aplicação Java.

### Tabela Fipe

A Tabela Fipe é uma referência nacional para os preços médios de veículos no mercado brasileiro. É amplamente utilizada por consumidores e profissionais do setor automotivo para consulta de valores de compra e venda de veículos.

### Tecnologias Utilizadas

- Java — Linguagem principal utilizada no desenvolvimento
- HTTP Client — Para realizar requisições à API da Tabela Fipe
- JSON — Para receber e manipular as respostas da API
- IntelliJ IDEA — IDE utilizada no desenvolvimento

### Funcionalidades

- Consulta de preços de veículos na Tabela Fipe
- Manipulação e exibição de dados retornados pela API
- Estruturação do código visando manutenção e escalabilidade

### Desafio da Alura

Este projeto foi proposto como um desafio pela Alura para reforçar o aprendizado de consumo de APIs em Java. Foram explorados conceitos como requisições HTTP, manipulação de JSON e boas práticas de desenvolvimento.

---

## 🇺🇸 English

This project was developed as an Alura challenge to practice API consumption using Java, with the Brazilian FIPE Table as the data source. It demonstrates how to perform HTTP requests, handle JSON responses, and integrate external data into a Java application.

### FIPE Table

The FIPE Table is a national reference for average vehicle prices in the Brazilian market. It is widely used by consumers and automotive professionals to check vehicle purchase and sale values.

### Technologies Used

- Java — Main programming language used in the project
- HTTP Client — To perform requests to the FIPE API
- JSON — To receive and manipulate API responses
- IntelliJ IDEA — Development environment used

### Features

- Vehicle price lookup using the FIPE Table API
- Handling and displaying returned API data
- Code structured for maintainability and scalability

### Alura Challenge

This project was proposed as a challenge by Alura to reinforce API consumption concepts in Java. It explores HTTP requests, JSON handling, and good development practices.
